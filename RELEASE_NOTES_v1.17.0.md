# GK Studio v1.17.0 Release Notes

## DEC-0026 Rollback

- Atomic Update前の現行Exportを永続バックアップする機能を追加
- 検証済みバックアップから原子的に復元する機能を追加
- 復元直前の現行世代も保存し、再ロールバック可能にした
- 更新・復元の共通排他ロックを採用
- 不正バックアップやシンボリックリンクを拒否
- `rollback-export.php` CLIを追加


## 提出物配布ルール（統一運用）
- すべての提出物（監査報告・検査結果・修正計画・成果物・ログ・チェックリスト）は必ずZIP形式で配布する。
- 単体ファイルのみで正式提出してはならない。
- ZIPには関連ログ、SHA-256一覧、マニフェスト等を含める。
- リリースチェックではZIP作成・CRC・展開確認・SHA確認を実施する。
