# GK Studio v1.16.0 リリースノート

## DEC-0025 Atomic Update

- 検証済みExportだけを現行データへ切り替える`AtomicExportUpdater`を追加
- 候補の事前検証、同一ファイルシステム上のstaging複製、再検証、rename切替を実装
- 並行更新を防ぐ排他ロックを追加
- 不正候補や複製失敗時に現行Exportを維持
- CLI更新コマンドを追加
- Atomic Updateの正常系・拒否系回帰試験を追加

次工程はDEC-0026 Rollbackです。


## 提出物配布ルール（統一運用）
- すべての提出物（監査報告・検査結果・修正計画・成果物・ログ・チェックリスト）は必ずZIP形式で配布する。
- 単体ファイルのみで正式提出してはならない。
- ZIPには関連ログ、SHA-256一覧、マニフェスト等を含める。
- リリースチェックではZIP作成・CRC・展開確認・SHA確認を実施する。
