# DEC-0011 適用記録 — Data Integrity Validator

## 採用範囲

1. 公式22パス固定検証
2. 同一ファイル内の重複ID検証
3. クエスト・MODの横断重複ID検証
4. 確定済みID参照整合性検証
5. 自動E2E・異常系試験への接続

## 計画進行への効果

- 正式マスターデータ量産の主要な停止要因を解消。
- 存在しないID参照と重複IDをPHPゲーム起動前に停止。
- 公式22ファイルの欠落・任意化・追加を拒否。

## 保留事項

- 孤立データは現時点では警告化しない。
- 循環参照は循環可能な正式フィールドが確定していないため未実装。
- 新しい参照項目は `schemas/data-integrity-rules.json` に追加する。

## 識別子について

既存Decision TrackerではDEC-0003が戦闘計算式に割当済みのため、本件は既存の「DEC-0011 マスター参照整合性」を採用・実装済みに更新した。
