# DEC-0025 — Atomic Update

- 状態: 決定済・実装済
- 判断: 採用
- 決定日: 2026-07-23
- 方針: 更新候補Exportを事前検証し、現行Exportと同一ファイルシステム上の一時ディレクトリへ複製・再検証した後、renameで切り替える。
- 排他制御: 対象Export単位のロックファイルを使用し、並行更新を`UPDATE_LOCKED`で拒否する。
- 失敗時: 候補検証・複製・再検証の失敗では現行Exportに触れない。切替処理が失敗した場合は直前の現行Exportを元のパスへ戻す。
- セキュリティ: 候補・対象・複製中のシンボリックリンクを拒否する。
- 対象外: 永続バックアップ世代と運用者による任意時点への復元はDEC-0026 Rollbackで扱う。
