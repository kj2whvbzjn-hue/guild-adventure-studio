# ギルド冒険物語 開発スタジオ V4.7 安定化版 v1.1.0

## 修正

- 保存履歴が実データ保存後に追加され、直近履歴が残らない問題を修正
- LocalStorage容量不足や保存例外を検知し、日本語で案内する処理を追加
- 24時間ごとの自動バックアップを追加
- 節削除前にもバックアップを作成するよう変更
- プロジェクト作成・複製・切替時の保存処理を安全化
- 終了時保存で例外が発生しても画面全体を停止させないよう改善

## 互換性

- JSON構造、内部ID、既存LocalStorageキーは変更していません
- 日本語版 v1.0.0 の既存プロジェクトをそのまま利用できます


## 提出物配布ルール（統一運用）
- すべての提出物（監査報告・検査結果・修正計画・成果物・ログ・チェックリスト）は必ずZIP形式で配布する。
- 単体ファイルのみで正式提出してはならない。
- ZIPには関連ログ、SHA-256一覧、マニフェスト等を含める。
- リリースチェックではZIP作成・CRC・展開確認・SHA確認を実施する。
