# GK Studio v1.3.0

## フェーズ3：キャラクター・世界設定管理

- キャラクターに種族、年齢、口調、所属、初登場、加入、離脱、覚醒を追加
- 第一・第二・第三目的を追加
- 組織管理を追加
- 用語辞典を追加
- キャラクター間の関係管理を追加
- 章・節単位の時系列管理を追加
- 組織、人物関係、時系列対象の参照検証を追加
- v1.2.0以前のJSONを自動補完して読込可能


## 提出物配布ルール（統一運用）
- すべての提出物（監査報告・検査結果・修正計画・成果物・ログ・チェックリスト）は必ずZIP形式で配布する。
- 単体ファイルのみで正式提出してはならない。
- ZIPには関連ログ、SHA-256一覧、マニフェスト等を含める。
- リリースチェックではZIP作成・CRC・展開確認・SHA確認を実施する。
