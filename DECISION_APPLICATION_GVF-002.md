# GVF-002 採用・適用記録

## 決定
ゲームバランス検査を固定値のPHP実装ではなく、JSONルールで変更可能な設定駆動型として採用する。

## 適用範囲
- 能力値・HP・MP・経験値・レベル・消費MP・iLv・MOD予算の許容範囲
- ドロップ率合計と重複アイテム
- レベル経験値列の単調増加
- 装備項目間の比較制約
- Critical / Warning / Info分類と100点満点スコア
- JSON / HTMLレポート

## 運用方針
正式Exportが空の場合はPASSとする。実データが存在し、対象フィールドが定義された時点でルールを評価する。許容値は schemas/balance-validation-rules.json で管理する。
