# ギルド冒険物語 開発スタジオ v1.2.0

## シナリオ管理機能
- 章・節・シーン・会話の制作状態を管理
- 節に目的、開始状態、終了状態、重要事項を追加
- 1章あたり最大20節を制御
- シナリオツリー検索を追加
- 章数、節数、シーン数、節の確定率を表示
- 既存JSONを読み込む際に新項目を自動補完

## 互換性
内部IDと既存フィールドは維持しており、v1.1.0以前のプロジェクトデータを読み込めます。


## 提出物配布ルール（統一運用）
- すべての提出物（監査報告・検査結果・修正計画・成果物・ログ・チェックリスト）は必ずZIP形式で配布する。
- 単体ファイルのみで正式提出してはならない。
- ZIPには関連ログ、SHA-256一覧、マニフェスト等を含める。
- リリースチェックではZIP作成・CRC・展開確認・SHA確認を実施する。
