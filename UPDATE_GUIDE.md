# v0.3.0 から v0.4.1 への更新手順

1. GitHubの `guild-adventure-studio` リポジトリを開く
2. `Add file`
3. `Upload files`
4. このフォルダ内のファイルをすべてドラッグする
5. `Commit changes` を押す
6. 1～2分待つ
7. GitHub Pagesを再読み込みする

## 更新後に古い画面が出る場合

Service Workerやブラウザキャッシュが残っている可能性があります。

Windows:
- Ctrl + F5

iPhone:
- Safariのページを閉じて開き直す
- 改善しない場合は、ホーム画面の古いアイコンを削除して再追加

旧バージョンで端末保存したデータは、同じGitHub Pages URLを使う限り自動的に読み込まれます。
