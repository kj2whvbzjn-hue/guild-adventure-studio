# Version 4.3 更新手順

1. 現在のStudioで重要なプロジェクトをJSON出力
2. ZIPを展開
3. リポジトリの Add file → Upload files
4. 全ファイルをアップロード
5. Commit changes
6. GitHub Pagesを1～2分後に再読込

## 古い画面が表示される場合

Windows:
Ctrl + F5

iPhone:
Safariを完全に閉じて開き直す。
改善しない場合はホーム画面アイコンを削除し、Safariから再度ホーム画面に追加する。

## 初回起動時

旧端末データは自動的にProjectsへ移行されます。
Explorerの `Projects` から確認できます。
