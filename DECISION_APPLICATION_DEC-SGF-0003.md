# Decision Application — DEC-SGF-0003

## Decision

Built-in automatic scenario generation is frozen. The SGF name and implementation baseline are retained but redefined as the **Scenario Support Framework**.

## Adopted use

- Chapter/Section/Event/Dialogue design forms
- Prompt Builder and copy-ready writing briefs
- Copy-and-paste workflow with ChatGPT or other external tools
- Manual draft intake
- Story Preview
- Consistency validation
- Candidate Revision comparison
- Human approval and Export

## Frozen use

- Built-in AI generation
- API connector
- Automatic regeneration
- Automatic quality scoring
- Automatic approval or merge

## Data safety

All imported or pasted results must be stored as candidate Revisions. Current data, LOCKED nodes, protected fields, approvals, and milestones cannot be changed directly.

Formal Build 328.
